Aims of this assignment:

Create a responsive layout, which:

- is built mobile first 
- at mobile resolution only displays the first two columns, that are pink and red respectively
- when scaled beyond 641px wide, using media queries, all columns will be displayed side by side
- extends no more than 1140px wide
- at larger resolution, from right to left, columns are colored blue, purple, grey brown
- created with valid HTML and CSS
- includes the necessary file structure
